# Car Parking Python Project

![Car Parking](https://via.placeholder.com/600x200?text=Car+Parking+Project)

## Project Overview
This Python project simulates a **Car Parking System** with entry and exit sensors, displays, and logging functionality. The system allows you to track vehicles entering and leaving the parking lot while updating display boards with real-time information about available bays.

---

## Features
- **Entry & Exit Sensors:** Automatically detect incoming and outgoing vehicles.
- **Displays:** Show real-time updates for available parking bays and other information.
- **Logging:** Keep a record of all vehicle entries and exits with timestamps.
- **Capacity Management:** Tracks the number of cars in the parking lot and prevents overflows.
- **Config File:** Load and save parking configuration using JSON.

---

## Technology Stack
- **Language:** Python 3.x
- **Modules:** pathlib, datetime, json, random
- **Concepts:** Object-Oriented Programming (OOP), Abstraction, Inheritance, File Handling
